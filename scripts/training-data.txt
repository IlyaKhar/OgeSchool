# Примеры диалогов для обучения

## Пример 1: Решение задачи по математике
Пользователь: Как решить задачу: "Найдите значение выражения 2x + 3 при x = 5"
Ассистент: Для решения этой задачи нужно подставить значение x = 5 в выражение 2x + 3:
1. Заменяем x на 5: 2 * 5 + 3
2. Выполняем умножение: 10 + 3
3. Выполняем сложение: 13
Ответ: 13

## Пример 2: Объяснение темы
Пользователь: Объясни что такое квадратные уравнения
Ассистент: Квадратное уравнение - это уравнение вида ax² + bx + c = 0, где a, b, c - числа, и a ≠ 0.
Примеры: x² - 5x + 6 = 0, 2x² + 3x - 1 = 0.
Для решения квадратных уравнений можно использовать:
1. Формулу дискриминанта: D = b² - 4ac
2. Если D > 0, то два корня: x₁ = (-b + √D) / 2a, x₂ = (-b - √D) / 2a
3. Если D = 0, то один корень: x = -b / 2a
4. Если D < 0, то корней нет

## Пример 3: Мотивация
Пользователь: Я не понимаю алгебру, что делать?
Ассистент: Не переживай! Алгебра кажется сложной, но с практикой станет понятнее. Начни с простых задач и постепенно переходи к сложным. Решай по 3-5 задач каждый день, и через месяц увидишь прогресс. Если что-то непонятно - задавай вопросы, я помогу разобраться!
